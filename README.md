# [ChatGPT Clone](https://chat-gpt-clone.vercel.app/)

[ChatGPT](https://chat.openai.com/) clone app built with Next.js app router.

## Getting Started

```
pnpm i

pnpm dev
```

And open http://localhost:3000

## Tech stack

- FW: Next.js
- Language: TypeScript
- Styling: Tailwind CSS
- ORM: Prisma
- Authentication: Auth.js
- API: Express
